Quick Documentation

This page demonstrates how query string parameters can be captured and used with Google Tag Manager, and how to make a dynamic API request based on those parameters.

Functionality Walkthrough

  1. Enter a key-value pair in the input fields.
  2. Click the "Send API Request" button to trigger the API call.
  3. The API response is logged to the browser console, showing the query parameters sent.
